# Open Brain - Chrome Extension

Browser extension that adds **Save Chat** and **Load Context** to 9 AI web tools.

## Supported Services

- ChatGPT (chatgpt.com)
- Claude (claude.ai)
- Gemini (gemini.google.com)
- Perplexity (perplexity.ai)
- Grok (grok.com)
- DeepSeek (chat.deepseek.com)
- Mistral (chat.mistral.ai)
- Copilot (copilot.microsoft.com)
- Poe (poe.com)

## Install

1. Go to `chrome://extensions` in Chrome
2. Enable **Developer Mode** (top right)
3. Click **Load unpacked**
4. Select this `chrome-extension` folder

## Configure

1. Click the extension icon in your toolbar
2. Go to **Options**
3. Enter your OpenBrain API URL and API key

Default API URL: `https://your-project.supabase.co/functions/v1/brain-api`

## Usage

Once installed, you'll see two buttons on any supported AI chat:

- **Save Chat** - Extracts the current conversation and stores it in your brain
- **Load Context** - Search your brain and paste relevant memories into the chat
